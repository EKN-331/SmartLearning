📌 Catatan Penting:
• Baris pertama adalah header (nama kolom) — tidak diimpor
• Kolom A: isi dengan ID mapel (math, science, indonesian atau ID mapel kustom)
• Kolom G: isi hanya A, B, C, atau D (huruf kapital)
• Kolom H (explanation) boleh dikosongkan
• File harus format .xlsx atau .csv
• Soal bergambar hanya bisa ditambah manual melalui form
